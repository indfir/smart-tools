CARA UPLOAD KE INDFIR.COM (HOSTINGER)
======================================

File di folder ini:
  index.html    -> aplikasi PDF Tools (versi online)
  ai-proxy.php  -> proxy server untuk fitur AI (wajib ikut diupload)

Langkah:
1. Login ke hPanel Hostinger -> Websites -> indfir.com -> File Manager.
2. Masuk ke folder public_html.
   - Mau di halaman utama?      taruh langsung di public_html/
   - Mau di indfir.com/pdf?     buat folder "pdf" lalu masuk ke situ.
   - Mau subdomain pdf.indfir.com? buat subdomain dulu di hPanel,
     lalu upload ke folder subdomain tsb.
3. Upload KEDUA file (index.html dan ai-proxy.php) ke folder yang sama.
4. Buka alamatnya di browser. Selesai — tidak ada instalasi lain.

Catatan:
- Tools inti (merge, split, rotate, delete/extract pages, organize,
  compress, JPG<->PDF, watermark, number pages, TXT to PDF, scanner)
  memproses file BENERAN, 100% di browser pengunjung. File tidak
  pernah terkirim ke server.
- Fitur AI: setiap pengguna memasukkan API key-nya sendiri (tersimpan
  di browser masing-masing). JANGAN menulis API key milikmu ke dalam
  file — semua orang bisa membacanya.
- ai-proxy.php hanya meneruskan request ke provider AI yang ada di
  allowlist, bukan open proxy.
- Versi lokal (folder pdf-tools-app) tetap bisa dipakai seperti biasa
  di komputer, dengan ai-proxy.py.
